# HARTON Certified 認定対象事業者 通知メール文案（草案 v0.1）

**版**: 0.1（草案）
**作成日**: 2026-04-27
**作成セッション**: ⑤ HARTON Certified 認定運用責任者
**位置付け**: MASTER-PLAN §11.2 / §11.4 + INSTRUCTION-FROM-ROOT.md #7 連動の通知メール正本草案
**配信前必須プロセス**:
1. ① レビュー（戦略・表現・ブランド整合性）
2. 弁護士確認（オプトアウト・引用ルール・景品表示法・特定商取引法整合）
3. 代表承認 → 配信
**配信先**: ★3 以上達成 + 致命的 NG ゼロの認定対象事業者

## 0. 設計方針

| 観点 | 方針 |
|---|---|
| トーン | 中立・事実ベース・断定調（MASTER-PLAN §2.5）。誇張・営業感を排除 |
| 訴求 | 「課金で順位が買えるサービスではない」「機械検証による公正評価」を明示 |
| 透明性 | 評価項目・閾値・実施手順の全公開を `/methodology/` で行うことに言及 |
| オプトアウト | **冒頭で必ず提示**。「掲載拒否は理由不要・即時対応」を明文化 |
| 法的論拠 | 弁護士相談済の旨を本文に含める（代表確認後に反映） |
| 個人情報 | 法人情報・公開連絡先のみ収集、第三者提供なしを明記 |

## 1. メール件名（候補）

採用候補 3 種（① レビューで決定）:

1. **【HARTON Certified】貴社サイトを ★X 認定とさせていただきました（掲載拒否権あり）**
2. **【無料認定】HARTON Certified ★X — 貴社サイトの WEB 品質評価結果のご報告**
3. **HARTON Certified ★X 認定のご案内（金銭非依存・機械検証）**

## 2. 本文（共通骨格）

```
{{事業者名}} 御中
{{宛先担当者}} 様

突然のご連絡を失礼いたします。
HARTON Certified 認定運用責任者の {{担当者名}} と申します。

このたび、HARTON Certified（認定機関）にて貴社の公式サイト
（{{事業者URL}}）を機械検証による評価対象とさせていただき、
{{rating}}（{{rating_label}}）として認定する判定結果を得ました。

本通知は、以下を貴社にお知らせするためのものです：

  1. 認定の事実と評価スコア
  2. 認定バッジの提供（自社サイトに掲載可能）
  3. 当機関のサイト（certification.tcharton.com）への
     貴社情報掲載のお伺い
  4. **掲載をご希望されない場合のオプトアウト窓口**

────────────────────────────────────
■ オプトアウト（掲載拒否権）について（最初にご案内）
────────────────────────────────────

貴社が当機関への掲載をご希望されない場合は、以下の窓口より
24 時間以内に対応いたします。理由のご説明は不要です。

  オプトアウト窓口: https://certification.tcharton.com/opt-out/
  または、本メールへの「掲載辞退」のご返信のみで結構です。

掲載辞退いただいた場合、貴社の認定データは即時削除いたします。
今後の再掲載は貴社の明示的な申請がない限り行いません。

────────────────────────────────────
■ HARTON Certified とは（中立・機械検証・金銭非依存）
────────────────────────────────────

HARTON Certified は、WEB サイトの品質を 4 軸の機械検証で公正に
評価する地方発の独立認定機関です。以下を運用方針としています。

  ・課金で順位を買えるサービスではありません
  ・タベログ等の口コミサイトとも異なります
  ・★3 以上のサイトのみを掲載するポジティブセレクション方式
   （低評価サイトは掲載せず、評価結果も公開しません）
  ・評価項目・閾値・実施手順は全公開
   （/methodology/ にて）

評価方法の詳細は以下をご確認ください：
  https://certification.tcharton.com/methodology/

  認定日: {{認定日}}

  ─── 機械検証スコア ───
  総合スコア: {{score}} / 100
  格付け: {{rating}}（{{rating_label}}）

  4 軸スコア（並列独立評価 / 軸間重みなし）:
    A 基礎・身だしなみ      : {{axes.A}} / 100

    C AI 検索適応           : {{axes.C}} / 100

★ 区分は年次基準改訂で見直されます。月次再スキャンで変動が生じた

────────────────────────────────────
■ 認定バッジについて
────────────────────────────────────

掲載に同意いただける場合、以下から認定バッジ（SVG / PNG / HTML 埋込
コード）をダウンロードいただけます。バッジには改竄検出ハッシュと
検証 URL が埋め込まれており、第三者が認定の真偽を確認できます。

  バッジ取得 URL: https://certification.tcharton.com/businesses/{{slug}}/badge/

バッジの不正使用や、認定 ID と URL の併記なしでの使用は禁止です
（詳細はバッジページのガイドライン PDF をご参照ください）。

Google ビジネスプロフィールが未作成の場合、Service Area Business
モードで作成いただくと住所非公開のまま登録可能です。詳細は改善
ガイドをご参照ください。
```

## 3. プレースホルダ一覧（generate.js または手動置換）

| プレースホルダ | データソース | 例 |
|---|---|---|
| `{{事業者名}}` | `data/businesses.json` の `name` | ○○税理士事務所 |
| `{{宛先担当者}}` | 通知前の代表者調査 | 代表 / 店主 / 院長 など |
| `{{担当者名}}` | 配信時固定（大内 達也） | 大内 達也 |
| `{{事業者URL}}` | `data/businesses.json` の `url` | https://example.com/ |
| `{{業種}}` | `data/industries.json` の `label`（businesses.json の industry キー経由） | 税理士・会計士 |
| `{{認定ID}}` | generate.js で発行（例: `HC-2026-0001`） | HC-2026-0001 |
| `{{認定日}}` | `data/businesses.json` の `approved_at` | 2026-05-01 |
| `{{score}}` `{{rating}}` `{{rating_label}}` `{{axes.A-D}}` `{{must5}}` `{{critical_ng}}` | `data/businesses.json` の `scan` | 各種数値 |
| `{{slug}}` | `data/businesses.json` のキー | numazu-cafe-suzuki |
| `{{弁護士確認日}}` | 配信時固定 | 2026-05-XX |
| `{{連絡先}}` | 配信時固定 | certification@tcharton.com / 電話 |
