## 🔵 セグメント A：消費者（情報収集ユーザー）

### ペルソナ
- 沼津・三島で「失敗したくない」と慎重に検索する住民
- 旅行者で地元のお勧めを知りたい人
- 「ホームページが綺麗な店＝信頼できる」と感覚的に思う人

### Entry Points

| 入口 | 検索意図例 |
|---|---|
| Google 検索 | 「沼津 美容院 おすすめ」「三島 ラーメン 老舗」 |
| AI 検索 | Perplexity / ChatGPT / Gemini で同様クエリ |
| SNS シェア | X / Instagram で「HARTON Stella」タグ |
| 掲載店のサイト | バッジ →（クリック）→ stella.tcharton.com |
| メインサイト | tcharton.com の事例ページから |
| note | 「水/土の地域業種別レポート」記事から |

### Journey

```
1. 検索結果 → 業種ページ or 地域ページ着地
2. ランキング一覧を眺める（★区分・スコアが視認できる）
3. ★4 や ★5 を中心に詳細クリック
4. 個別事業者ページで詳細確認
   ・★ 区分・認定 ID
   ・4 軸スコア（レーダーチャート）
   ・サイトのスクリーンショット
   ・公式サイトリンク
5. 公式サイトへ離脱（HARTON Stella 役割完了）
```

### 必要な UX 機能

| 機能 | 重要度 |
|---|---|
| ★ 区分でフィルタ | ⭐⭐⭐⭐⭐ |
| 業種 × 地域でナビ | ⭐⭐⭐⭐⭐ |
| 公式サイトへの目立つ CTA | ⭐⭐⭐⭐ |
| 「他の優良店も見る」関連表示 | ⭐⭐⭐⭐ |
| サイト内検索 | ⭐⭐⭐ |
| 地図表示 | ⭐⭐ |

### KPI

| 指標 | 目標値（Phase 1）|
|---|---|
| **公式サイトへのクリック数** | 月 1,000+ |
| 業種ページ滞在時間 | 平均 2 分以上 |
| 直帰率 | 50% 以下 |
| 業種・地域ページ間の回遊率 | 30%+ |

## 🟢 セグメント B：掲載事業者（既掲載・受動的）

### ペルソナ
- HARTON からの認定通知メールで知った店主
- 業種仲間から「お前のところ載ってるよ」と聞いた経営者
- 自社が掲載されているか自分で検索した事業者

### Entry Points

| 入口 | きっかけ |
|---|---|
| 認定通知メール | HARTON からの自動送信 |
| 業種仲間の口コミ | 既掲載の同業者から |
| 自社業種ページ | 検索で自社を発見 |

### Journey

```
1. 通知メール / 業種ページ → 個別事業者ページ着地
2. 認定の事実確認（★区分・認定 ID・認定日）
3. 「認定バッジを取得」セクションを発見
4. バッジダウンロード（SVG / PNG / HTML スニペット）
5. 改善余地セクション → 「★4 → ★5 へ」のガイダンス確認
6. （任意）HARTON サービスへ誘導 → tcharton.com で改善依頼
```

### 必要な UX 機能

| 機能 | 重要度 |
|---|---|
| バッジ 1-2 クリックダウンロード | ⭐⭐⭐⭐⭐ |
| HTML 埋込みコード（clipboard コピー）| ⭐⭐⭐⭐⭐ |
| 「あなた向け」セクション（事業者向け案内） | ⭐⭐⭐⭐⭐ |
| 改善ヒントの階層表示（★3→★4→★5）| ⭐⭐⭐⭐ |
| 認定書 PDF 発行（押印風）| ⭐⭐⭐⭐（営業ツール）|

### KPI

| 指標 | 目標値（Phase 1）|
|---|---|
| **バッジダウンロード率** | 掲載店の 70%+ |
| 自社サイトへの **バッジ埋込み率** | 掲載店の 40%+（最重要・SEO 効果）|
| 改善ガイド閲覧率 | 掲載店の 50%+ |
| ★ 昇格申請数（任意制度）| 月 3-5 件 |

## 🟡 セグメント C：未掲載事業者（自社も載りたい）★最重要

### ペルソナ
- 業種仲間のバッジを見て羨ましいと感じた経営者
- 自社サイトに自信がある事業者
- WEB に投資意欲のある経営者
- 「うちのサイトもプロが評価したらどうなる?」が気になる人

### Entry Points

| 入口 | きっかけ |
|---|---|
| 業種仲間のバッジ | クリックで certification 着地 |
| 業種ページで自社不在 | 「あれ、うちは?」 |
| メディア記事 | 取材記事で HARTON Stella を知る |
| Google 検索 | 「HARTON Stella 申請」「優良 WEB 認定 静岡」 |

### Journey（最重要動線・MASTER-PLAN §5.2 Q3(a) ダイレクト）

```
1. 知った経路 → /apply/ ページ着地
2. 「認定の流れ」を確認
   ・申請 → 無料診断 → 評価 → 結果通知 → 掲載
3. 「私のサイトはどう評価される?」と疑問
4. 「無料診断はこちら」CTA をクリック
5. tcharton.com/contact/ または専用フォームへ送客 ★最重要
6. HARTON 代表との初回コンタクト
7. （改善が必要なら）有料サービス契約 → 改善 → 再評価 → 認定
```

### 必要な UX 機能

| 機能 | 重要度 |
|---|---|
| /apply/ で「希望を持たせる」設計 | ⭐⭐⭐⭐⭐ |
| 申請プロセスのビジュアル化 | ⭐⭐⭐⭐⭐ |
| 「無料診断」ボタン（tcharton.com 送客）| ⭐⭐⭐⭐⭐ |
| 「掲載までの期間目安」明記 | ⭐⭐⭐⭐ |
| 「掲載されない場合の改善方法」も提示（懐の深さ）| ⭐⭐⭐⭐ |
| 過去の認定事例（before/after）| ⭐⭐⭐ |
| 料金透明性（無料診断は無料、改善は別途）| ⭐⭐⭐⭐⭐ |

### KPI（**全 KPI の中で最重要**）

| 指標 | 目標値（Phase 1）|
|---|---|
| **tcharton.com 送客クリック数** | 月 50+ |
| **無料診断申込数** | 月 10+ 件 |
| **HARTON サービス契約数** | 月 2-3 件 |
| /apply/ 滞在時間 | 平均 3 分以上 |
| 申請フォーム submit 数 | 月 5+ 件 |

## 🟣 セグメント D：メディア（取材・記事執筆）

### ペルソナ
- 静岡新聞・伊豆新聞の記者
- 地域経済誌（しずおかチェック等）のライター
- 業界専門誌の記者
- ローカルブロガー
- Phase 後半: 大手 IT メディアの記者

### Entry Points

| 入口 | きっかけ |
|---|---|
| プレスリリース | HARTON 主導の発表 |
| 業界知人の紹介 | 既掲載店主からの紹介 |
| 月次ランキング更新 | 「先月の TOP 10」note 記事 |
| HARTON 代表の note | 業界記事の執筆ネタ |

### Journey

```
1. プレスリリース/紹介 → /press/ ページ着地（新規ページ）
2. 引用可能データを確認
   ・月次ランキング（CSV ダウンロード可）
   ・代表写真・ロゴ高解像度
   ・引用条件・クレジット表記方法
   ・取材実績一覧
3. 取材依頼 → /contact/ で「メディア取材」を選択
4. 代表とのインタビュー → 記事化
5. 記事公開 → 被リンク・引用獲得
```

### 必要な UX 機能（新規ページ /press/）

| 機能 | 重要度 |
|---|---|
| 引用素材ダウンロード（ロゴ・写真・データ）| ⭐⭐⭐⭐⭐ |
| 月次ランキングの CSV / JSON 提供 | ⭐⭐⭐⭐ |
| クレジット表記ルール明記 | ⭐⭐⭐⭐ |
| 取材実績一覧 | ⭐⭐⭐ |
| メディア向け問合せフォーム | ⭐⭐⭐⭐ |
| プレスリリース履歴 | ⭐⭐⭐ |

### KPI

| 指標 | 目標値（Phase 1）|
|---|---|
| **メディア掲載数** | 年 5+ 件 |
| **被リンク（引用）数** | 月 3+ 件 |
| 取材依頼数 | 月 1+ 件 |
| Wikipedia 記事化 | 1 件（Phase 4 までに）|
