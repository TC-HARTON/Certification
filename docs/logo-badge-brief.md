## 0. プロジェクト概要

HARTON Stella（stella.tcharton.com）は、静岡県東部の優良 WEB サイトを公正に評価・認定する独立機関。SPEC v3.2 という独自規格と、4 軸スキャナーによる機械検証で業種別・地域別ランキングを公開する。

## 1. ブランド原型

- ミシュランガイド × Webby Awards × ISO 認証 のハイブリッド
- 「静かな権威」「ガイドブック」「認定状」のトーン
- 派手でなく、見る人が分かる人にだけ重みが伝わる

## 2. カラーパレット（確定）

|---|---|---|

## 3. ロゴ方向性（案 E ハイブリッド型 採用）

### 構造

```
円形リング（紺）
  ├─ 中央: ★（金）
  │   - ★の上下に "H"+"C" モノグラム
  │   - ★ から放射状の光線（金、微細）
  └─ リング内側に円弧で
      "HARTON · CERTIFIED · 静岡県東部 · "
全体下に "EST. 2026"
```

### 印象の要件

| ✅ 必須 | ❌ NG |
|---|---|
| 灯台メタファー（光線・★）| 観光地ロゴ風（ゆるキャラ・親しみすぎる）|
| 紋章感（円形リング）| IT スタートアップ風（無機質・テック前面）|
| モダン（モノグラム）| アワード風の派手さ（ゴテゴテ・装飾過多）|
| ガイドブック感 | レガシーすぎる威厳（時代遅れ）|

### 必ず伝わるべき 3 つの印象

1. 「これは認定の証だ」（official mark）
2. 「これは技術が裏付けている」（technical credibility）
3. 「これは地域に根差している」（local + universal）

---

## 4. ロゴの 3 階層

### 階層 1: シンボルマーク（Symbol）

```
円形リング + ★ + モノグラム
```
- 単独使用可（Apple のリンゴマーク的に）
- 最小サイズ: 16×16 px（ファビコン）まで識別可能
- 単色版もあり（紺一色、金一色、白抜き）

### 階層 2: ロゴタイプ（Wordmark）

```
HARTON CERTIFIED
```
- フォント: serif 系（Playfair Display や Cormorant Garamond 系統）
- weight: 500-600（重厚だが堅すぎない）
- カーニング: ゆとりめ（高級感）
- 全大文字 / Title Case のバリエーション

### 階層 3: ロックアップ（Lockup）

```
[Symbol] HARTON CERTIFIED
         静岡県東部 公正 WEB 認定
```
- Symbol + Wordmark + Tagline の組み合わせ
- 横並び型 / 縦並び型 / 中央揃え型 の 3 バリエーション

## 6. 形状バリエーション

| 形状 | 用途 | サイズ |
|---|---|---|
| 円形（メイン）| サイト・名刺・印刷物 | 200×200px / SVG |
| 盾型（アワード風）| プレスリリース・記念物 | 200×260px |
| 横長バー | サイトフッター・メールシグネチャ | 400×80px |
| 縦長プレート | サイドバー・ステッカー | 100×280px |
| 正方形（SNS）| プロフィール画像 | 400×400px |
| ファビコン | サイトタブ用 | 32×32px / 16×16px |

→ **6 形状 × 3 ★区分 = 18 バッジ**を提供する設計。

## 10. 不正使用防止

| 仕組み | 内容 |
|---|---|
| 認定 ID + URL 併記必須 | バッジには必ず認定 ID と検証 URL を埋込み |
| SVG メタデータ | XML 内に認定情報・発行日・改竄検出ハッシュを埋込み |
| 検証ページ | 認定 ID をサイトで検索→真偽判定可能 |
| 期限切れ表示 | 期限超過すると別バッジに自動切替 |
| ガイドライン PDF | 使用条件・改変禁止・印刷時の最小サイズ等を明記 |

## 12. 期限管理

```
認定日 → +12 ヶ月 → 再スキャン → 評価維持 / 変動 → 新バッジ発行
```

- 月次再スキャンで★区分が変動した場合、即座に新バッジ発行
- 旧バッジは「期限切れ」として無効化（事業者に通知）
- 過去の認定履歴は事業者ページに「沿革」として残す

## 14. 制作経路の選択肢

### 選択肢 A: プロデザイナー外注

| メリット | デメリット |
|---|---|
| 高品質・オリジナリティ | コスト 30-100 万円 |
| ブランド構築の伴走 | 期間 2-4 週間 |
| 商標化までの一貫サポート | デザイナー選定が要 |

→ 推奨: ブランドの長期投資として正攻法

### 選択肢 B: AI 画像生成（Midjourney / DALL-E 等）

| メリット | デメリット |
|---|---|
| 即日・低コスト（〜数千円）| 商標化に弱い |
| 多数のバリエーション試せる | 同一サービスでの再現性 |
| プロトタイプには最適 | プロの polish には届かない |

→ 推奨: Phase 0 のパイロット段階で使う / 最終形を外注

### 選択肢 C: 自社制作

| メリット | デメリット |
|---|---|
| ブランドへの理解最大 | 工数大（デザインスキル要）|
| 反復改善容易 | 客観性の欠如リスク |

### 推奨：段階的アプローチ

```
[Phase 0]
  AI 生成で 5-10 案を試作（1 日）
  ↓ 1 案を選定 + 自社で微調整
  ↓ パイロット 30 件で運用、フィードバック収集
[Phase 1 移行時]
  選定案を **プロデザイナーに「リブランディング」依頼**
  ↓ ロゴガイドライン PDF 完備
  ↓ 商標登録（必要なら）
[Phase 2-4]
  確定したロゴで全国展開
```
