弁護士 の WEB 品質評価ポイント

  • Trusted Types CSP 適用(法律相談フォームの XSS 経路完全遮断)
  • プライバシーポリシー充実度(個人情報保護法準拠 / WCAG 2.2 AA フォーム)
  • JSON-LD LegalService + ProfessionalService の重ね合わせ実装
  • 公式 SSL 証明書(EV または OV 推奨 / 期限切れ即致命的 NG)
  • AI 検索適応 (GEO §G-2 公的リンク 5+ 件 / 弁護士法 e-Gov 引用)

弁護士 WEB 品質チェックリスト

  1. HTTPS + HSTS preload 登録済 / SSL 期限切れなしか
  2. 法律相談フォームが Trusted Types + CSP form-action で保護されているか
  3. プライバシーポリシーが個人情報保護法 + 弁護士法に準拠しているか
  4. JSON-LD で LegalService + ProfessionalService が両方記述されているか
  5. WCAG 2.2 AA: フォーム input に label for 関連付け済か
  6. <blockquote cite> で関連法令 (e-Gov 弁護士法) を引用しているか
  7. 顧問弁護士 / 民事訴訟 / 刑事弁護 等のサービスが Schema.org Service で構造化されているか
  8. GBP 連携 + areaServed 記述で地域 SEO 強化されているか

弁護士 サイトで致命的 NG となる項目は？

HTTPS 非対応 / SSL 証明書無効 / WordPress 管理面露出 (wp-login.php / xmlrpc.php) / CMS バージョン情報露出 の 4 項目が致命的 NG。一発除外条件。

弁護士 の Schema.org 推奨タイプは？
