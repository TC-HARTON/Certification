クリニック・診療所 の WEB 品質評価ポイント

  • 診療予約フォームの HTTPS + Trusted Types CSP 適用(医療個人情報 / 致命的 NG 直結)
  • Schema.org MedicalClinic + MedicalBusiness + medicalSpecialty 構造化
  • 診療科目ページの Schema.org MedicalSpecialty 値設定 (Pediatrics / Dermatology 等)
  • 診療時間 openingHoursSpecification の machine-readable 表記
  • WCAG 2.2 AA 準拠 (患者多様性対応)

クリニック・診療所 WEB 品質チェックリスト

  1. HTTPS + HSTS preload 登録済 / SSL 期限切れなしか
  2. 診療予約フォームが Trusted Types + CSP form-action self で保護されているか
  3. JSON-LD で MedicalClinic + medicalSpecialty が記述されているか
  4. 診療科目が Schema.org MedicalSpecialty で machine-readable か
  5. 緊急時連絡先が視認しやすく配置されているか
  6. WCAG 2.2 AA: フォーム input の label 関連付け・コントラスト比 4.5:1 以上か
  7. 医療法 (e-Gov) を <blockquote cite> で引用しているか
  8. GBP 連携 + availableService + openingHours が構造化されているか

よくある質問

Phase 0.5 静岡県 5 都市 で信頼できる クリニック・診療所 のサイトを見分けるには？

HARTON Certified は クリニック・診療所 サイトを 4 軸（基礎・防御・AI 検索・経営）で機械検証し、MedicalClinic Schema.org 適合 + HTTPS + Trusted Types CSP の 3 条件を必須とする。Phase 0.5 静岡県 5 都市 の認定状況は本ページ上部で確認可能。

Phase 0.5 静岡県 5 都市 の クリニック・診療所 サイトで HARTON Certified 認定件数は？

Phase 0.5 静岡県 5 都市の クリニック・診療所 サイトを 69 件機械検証 (5 都市集計) し、★ 認定 0 件 / 業界最高点 53 点 / 致命的 NG 33.3% (2026-05-02 時点)。詳細は静岡県 WEB 品質業界レポート 2026 Q2 (4-6 月号)に公開。

クリニック・診療所 サイトの Schema.org 推奨タイプは？

MedicalClinic + MedicalBusiness + LocalBusiness の 3 層を推奨する。診療科目を MedicalSpecialty で構造化し、availableService に保険診療内容を記述することで AI 検索引用率が向上する。

クリニック・診療所 サイトで致命的 NG となる項目は？

HTTPS 非対応 / SSL 証明書無効 / 診療予約フォームの CSP 未設定 / CMS バージョン情報露出 の 4 項目が一発除外条件。医療個人情報を扱う機関として特に致命的。

クリニック・診療所 サイトのアクセシビリティ対応は必要か？

患者の多様性（高齢者・障害者等）から WCAG 2.2 AA 準拠が強く推奨される。フォーム label 関連付け・コントラスト比 4.5:1 以上・キーボード操作対応・緊急連絡先の視認性確保が評価対象。
